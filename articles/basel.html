バーゼル問題

はじめに

有名な定理としてすべての自然数の2乗の逆数の和は円周率の二乗を6で割ったものと等しいというものがあり、バーゼル問題と呼ばれています.証明は頑張れば高校範囲で実現することもできますがここではEulerが発見したマクローリン展開を使った証明の方法を超ざっくりと説明します.そのあとでそれを利用した円周率の計算方法を掲示します.

超雑な証明

  1. f(x)=sin(x)/xとする
  2. f(x)をマクローリン展開する
  3. f(x)=0の解の周期性に注目して因数分解する
  4. 2の結果と3の結果におけるx^2の項のの係数を比較すると求めたい級数が得られる

バーゼル問題を利用した円周率の計算方法

Pythonで簡単に実装できます

                
                    import math

                    def calculate_pi(n):
                        pi_sum = 0
                        for i in range(1, n+1):
                            pi_sum += 1/(i**2)
                        pi = math.sqrt(pi_sum*6)
                        return pi
                    
                        print(calculate_pi(1000000))

実際にはこの計算方法は収束が非常に遅いです.100万項まで計算しても小数点以下5桁程度の制度しか出ません.代替としてコンピューターで円周率を計算する場合にはマチンの公式などが使われることが多いです.
